# GNSS Java Documentation

Welcome to the documentation site for **GNSS Java**, a Maven-based Java project for working with GNSS/NMEA streams and related NTRIP/RTCM workflows.

## What you'll find here

- Setup and local development instructions
- High-level architecture and package layout
- Focused guides for NMEA and NTRIP/RTCM usage
- Generated Java API documentation (JavaDoc)
